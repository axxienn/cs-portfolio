Step 2: Javascript

To start off a javascript(JS) file, gather elements from HTML. Here is how I did mine:

  • The keyword var, or variable, is used for storing data values in JS
  • Every var must be followed by an identifier
    • The identifiers from above would be "input", "select", "input2", "button", "ans", and "answer"
    • An identifier should be be unique to prevent different variables operating the same function
  • The equal sign (=) in JS is known as an assignment operator. This functions by assigning a value to the variable.
    • Take the first variable from above for example, document.getElementById("input") is the value that is assigned to the var input
  • document.getElementById(" ") is used when you want to get information from another file
    • The Document Object Model (DOM) is a common method to control the structure as well as style of a HTML page, and getting an element by ID is just one way to do it
    • camelCase is a JS style of capitalizing the first letter of every word, except for the first word. The purpose of camelCase is to allow readers to recognize long unspaced words easier
