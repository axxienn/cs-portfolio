Motherboard

Motherboard is a computer's central in which allows different parts of the computer to be gathered at one place and communicate with each other. It looks like a mini contraption on plastic board. The different components potentially would include a CPU, RAM, multiple USB ports, ect. It is also known as "the heart of the computer".

CPU

CPU stands for Central Processing Unit, or the brain of the computer. It is a computer's primary component and is responsible for translating your instructions or commands into action. And in order to carry out the inputs, the CPU also contains other units such as the arithmetic logic unit (ALU) and the control unit (CU).

RAM

RAM is an accronym for random access memory. It stores memories temporarily and use it to carry out certain instruction on hand. The "random access" is the result of its ability to store information from any location.
